Через веб-интерфейс GitHub

    Войдите в аккаунт GitHub

        Перейдите на github.com

        Нажмите "Sign in" (если не авторизованы)

    Создание репозитория

        В правом верхнем углу нажмите + → New repository

        Или перейдите по прямой ссылке: github.com/new

    Заполните информацию:

        Repository name - название репозитория

        Description - описание (необязательно)

        Public/Private - выберите видимость

        Initialize this repository with:

            README file (рекомендуется)

            .gitignore (выберите шаблон)

            license (выберите лицензию)

    Нажмите "Create repository"

 Через веб-интерфейс GitHub:

1. **Перейдите на главную страницу репозитория**, который хотите удалить
2. Нажмите на вкладку **Settings** (Настройки)
3. Прокрутите страницу вниз до раздела **Danger Zone** (Опасная зона)
4. Нажмите на кнопку **Delete this repository** (Удалить этот репозиторий)
5. Вам потребуется:
   - Ввести название репозитория для подтверждения
   - Нажать кнопку **I understand the consequences, delete this repository**

## Важные моменты:

⚠️ **Внимание!**

- Удаление репозитория **необратимо**
- Будут удалены все ветки, коммиты, issues, pull requests и вики
- Если репозиторий был форком, исходный репозиторий не пострадает
- Рассмотрите возможность сделать backup перед удалением

## Альтернативные варианты:

Если не хотите полностью удалять:

1. **Сделать приватным** - в настройках изменить видимость на Private
2. **Перенести в другой аккаунт** - Transfer ownership в настройках
3. **Архивировать** - можно добавить пометку в README, что проект заархивирован
